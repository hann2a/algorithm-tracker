[BOJ 1234] Array Rotation

Implementation · 2026-01-07

LINK

https://boj.kr/1234

SUMMARY

Rotate an array clockwise. Solved using deque.

CODE

from collections import deque

def rotate(arr, k):
    d = deque(arr)
    d.rotate(k)
    return list(d)
